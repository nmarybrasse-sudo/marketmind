# MarketMind — Competitor Intelligence

## Project Overview
Purpose: Browser-based competitor analysis dashboard
Audience: Non-technical business users.

## Tech Stack
- Single HTML file (index.html) — no build tools
- Vanilla JS, Chart.js via CDN
- No backend, no login, no database, no API

## Style
- Color: navy + white + accents
- Tone: Professional, Bloomberg-terminal feel

## Rules
- Never break the single-file constraint
- Test in browser before declaring done
