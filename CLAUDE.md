# Data Analysis Guidelines

When analyzing data:

1. **Inspect the data first** — show shape, dtypes, and missing values before anything else
2. **State all assumptions explicitly** before proceeding
3. **Lead with the headline insight**, then support it with numbers
4. **Every chart must answer one specific business question** — label it clearly
5. **Save all outputs** (charts, reports, tables) to an `output/` folder
